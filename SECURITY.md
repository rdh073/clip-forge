# Security Policy

## Reporting

Please report security issues privately via the GitHub Security Advisories
form on this repository:

  https://github.com/rdh073/clip-forge/security/advisories/new

Do not open a public issue for security reports.

## Scope

This policy covers the `clip-forge` plugin code itself, not its third-party
dependencies. For issues in dependencies (`@mediapipe/tasks-vision`, `ffmpeg`,
`yt-dlp`, MCP servers, etc.), please report upstream.

## Response

We aim to acknowledge reports within **7 days** and provide a fix or
mitigation within **30 days** for confirmed issues. Reporters will be
credited in the release notes for any disclosed advisory unless they
prefer to remain anonymous.
